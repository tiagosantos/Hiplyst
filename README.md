# Hiplyst

Rails API + ReactJS development with based on this tutorial https://medium.com/@bruno_boehm/reactjs-ruby-on-rails-api-heroku-app-2645c93f0814

* Ruby 2.5.1p57

* Rails 5.1.6

* Configuration
  cd list-app
  rails s -p 3001
  
  cd client 
  yarn start
  
* Database creation
  rails db:migrate

* Database initialization
  rails db:seed
